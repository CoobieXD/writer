# Писец

Онлайн-генератор рукописных конспектов. Превращает цифровой текст в изображение с имитацией настоящего рукописного почерка — с использованием отсканированных спрайтов реальных букв.

Работает полностью в браузере, без серверной части.

**[Live Demo](https://coobiexd.github.io/writer/)**

## Поддерживаемые символы

Кириллица, латиница, цифры, пунктуация, греческий алфавит, математические символы, спецсимволы, аббревиатуры (автоматическое определение).
